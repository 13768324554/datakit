Module Datakit_github.Snapshot

module Snapshot: sig .. end

GitHub snapshot


type t
The type for GitHub snapshot.
val pp : t Fmt.t
pp is the pretty-printer for snapshots.
val empty : t
The empty snapshot.
val is_empty : t -> bool
is_empty t is true if t is Datakit_github.Snapshot.empty.
val create : repos:Datakit_github.Repo.Set.t ->
       commits:Datakit_github.Commit.Set.t ->
       status:Datakit_github.Status.Set.t ->
       prs:Datakit_github.PR.Set.t ->
       refs:Datakit_github.Ref.Set.t -> t
create ~repos ~commits ~status ~prs ~refs is a new snapshot t with repositories reps, commits commits, pull-requests prs, build statuses status and Git references refs.
val compare : t -> t -> int
compare is the comparison function for snapshots.
val union : t ->
       t -> t
union x y is the union of the snapshots x and y.
val prune : t -> t
prune t is t where all the objects related to closed PRs have been removed.

Diffs


type diff
The type for snapshot diffs.
val diff : t ->
       t -> diff
diff x y is the difference between x and y.

Elements


val elts : t -> Datakit_github.Elt.Set.t
elts t is the collection of elements of t.
val repos : t -> Datakit_github.Repo.Set.t
repos t are t's repository.
val prs : t -> Datakit_github.PR.Set.t
prs t are t's pull-requests.
val commits : t -> Datakit_github.Commit.Set.t
commits t are t's commits.
val status : t -> Datakit_github.Status.Set.t
status t are t's build status.
val refs : t -> Datakit_github.Ref.Set.t
refs t are t's Git references.
val with_elt : Datakit_github.Elt.t ->
       t -> t
with_elt e t it t with the element e added.
val with_elts : Datakit_github.Elt.Set.t ->
       t -> t
with_elts is like Datakit_github.Snapshot.with_elt but for a collection of elements.
